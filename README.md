# OgataChannel with AI Official Site

静的HTML/CSS/JSで作った公式サイトのたたき台です。

## 差し替えるもの

- `assets/key-visual-2.png`: TOPの固定背景に使うキービジュアル2
- 実素材は `assets/source/` に展開済みです。サイトで使う素材は英字ファイル名にコピーしています。
- `index.html` 内の外部リンク: YouTube / pixiv / BOOTH / X の正式URL
- `contact@example.com`: 公開用メールアドレス
- `History` と `Works` の本文: 公開日、衣装名、作品タイトル、制作メモ

## 開き方

`index.html` をブラウザで開けば確認できます。
